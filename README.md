# 인테리어 견적 계산기

실제 인테리어 현장 견적서 및 자재 단가표를 기반으로 만든 **웹 기반 인테리어 자동 견적 계산기**입니다.

## 미리보기

항목을 선택하고 수량을 입력하면 소계 / 부가세 / 합계 / 평당 단가가 자동으로 계산됩니다.  
마루, 도배 등 **평(평당) 단위 항목**은 평수 입력값이 바뀌면 수량이 자동 반영됩니다.

## 사용 방법

### 로컬 실행

```bash
git clone https://github.com/YOUR_USERNAME/interior-calculator.git
cd interior-calculator
# 브라우저에서 index.html 열기 (서버 없이 바로 실행 가능)
open index.html
```

### GitHub Pages 배포

1. GitHub에서 새 레포지토리 생성
2. 파일 업로드 또는 `git push`
3. Settings → Pages → Branch: `main` / `root` 선택 → Save
4. `https://YOUR_USERNAME.github.io/interior-calculator` 접속

## 주요 기능

- **평수 자동 연동** — 평수 입력 시 마루·도배 등 평당 항목 수량 자동 업데이트 (`pyAuto: true`)
- **업체이윤 자동 계산** — 다른 항목 합계 기준 % 자동 반영 (`pct: 0.1`)
- **라이트/다크 모드** — OS 설정 자동 감지
- **모바일 반응형** — 스마트폰에서도 사용 가능
- **부가세 별도 계산** — 소계 × 10% 자동 표시

## 단가 업데이트 방법

`data.js` 파일에서 해당 항목의 `p` 값을 수정하거나, 새 항목을 추가합니다.

```js
{ id:"f1", n:"강마루 강그린프로", d:"LX 95x800x6.0T 시공비포함", u:"평", p:87500, pyAuto:true },
//                                                                              ↑ 이 값을 수정
```

새 카테고리를 추가하려면 `DATA` 객체에 새 키를 추가하면 됩니다.

```js
"새카테고리": { items: [
  { id:"new1", n:"항목명", d:"설명", u:"단위", p:100000 },
]},
```
